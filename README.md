# Ewaz Fakor - Personal Website

Welcome to my personal portfolio and blog website. This repository contains my professional profile, projects, work experience, links, and blog articles.

## Repository Structure

```
ewazfakor.github.io
│
├── index.html              # Main homepage
├── links.html              # Useful links and resources
├── README.md               # This file
│
├── blog/                   # Blog section
│   ├── index.html          # Blog main page with article list
│   └── first-post.html     # Sample blog post
│
└── assets/                 # Static assets
    ├── css/
    │   └── style.css       # Global stylesheet
    ├── images/
    │   └── avatar.jpg      # Profile avatar
    └── icons/
        └── favicon.ico     # Website favicon
```

## Design System

### Typography
- **Font:** Source Serif Pro (serif)
- **Base Size:** 14pt
- **Heading Font Weight:** 700 (bold)
- **Body Font Weight:** 400 (regular)
- **Line Height:** 1.6 for body, 1.8 for posts

### Colors
- **Primary Text:** #1a1a1a (dark gray/black)
- **Links:** #0096FF (bright blue)
- **Link Hover:** #0078CC (darker blue)
- **Secondary Text:** #555-#666
- **Borders:** #eaeaea (light gray)
- **Background:** #fff (white)

### Layout
- **Container Max Width:** 2560px
- **Side Margins:** 120px (desktop), 20px (mobile)
- **Mobile Breakpoint:** 600px and below

### Visual Elements
- **Bullet Points:** ▪ (small square, 1.2em size)
- **Item Indentation:** 28px
- **Section Borders:** 1px solid #eaeaea
- **Border Radius:** 50% for avatar

## Navigation

All pages include consistent navigation:
- Home link (returns to index.html)
- Blog link (to blog/index.html)
- Links page (to links.html)
- External links (Medium, Instagram, GitHub, etc.)

## Responsive Design

The site is fully responsive with breakpoint at 600px:
- **Desktop:** 14pt font, 120px margins, 252x252px avatar
- **Mobile:** 13pt font, 20px margins, 150x150px avatar
- Navigation and spacing adapt for smaller screens

## Persian Text

Name and Persian text use Vazirmatn font with:
- Right-to-left (rtl) direction
- 1em font size (matches body)
- Proper Unicode support

## Getting Started

1. Clone this repository
2. Ensure all files are in the correct directory structure
3. Update content in each HTML file with your information
4. Replace avatar.jpg with your profile photo (252x252px)
5. Add favicon.ico to assets/icons/ folder
6. Customize colors in style.css if desired

## File Naming Conventions

- All HTML files use lowercase naming
- CSS file is singular: `style.css` (not `styles.css`)
- Images use lowercase with hyphens: `avatar.jpg`
- Blog posts use descriptive slugs: `first-post.html`

## Browser Support

- Modern browsers (Chrome, Firefox, Safari, Edge)
- Mobile browsers (iOS Safari, Chrome Mobile)
- Responsive design adapts to all screen sizes
